# Sistema de Gestión con Control de Roles

## Descripción
Sistema completo de gestión con autenticación de usuarios, control de roles y dashboard interactivo.

## Características
- **Login con fondo de estrellas**: Interfaz transparente con vista de cielo nocturno
- **3 Roles de usuario**: Superusuario, Admin y Usuario (2 usuarios por rol)
- **Dashboard interactivo**: Tarjetas de estadísticas, calendario, gráficos y panel de documentos
- **Topbar**: Barra superior negra con notificaciones interactivas y perfil de usuario
- **Sidebar**: Panel lateral colapsable con navegación y logo
- **Diseño**: Fuente Times New Roman, botones azules, logo tipo cubo de Geometry Dash

## Componentes del Dashboard
1. **4 Tarjetas de estadísticas**: Roja, azul, verde y amarilla
2. **Calendario interactivo**: Con fechas célebres y navegación
3. **Gráfico de actividad**: Muestra actividad del usuario
4. **Panel de documentos**: Estado, nombre y fecha de modificación

## Páginas Implementadas
1. **Login** (index.html): Autenticación de usuarios
2. **Dashboard** (dashboard.html): Página principal con estadísticas
3. **Gestión de Usuarios** (users.html): Lista completa de usuarios del sistema
4. **Contactos** (contacts.html): Directorio de contactos
5. **Archivos** (files.html): Sistema de gestión de archivos con subir/descargar
6. **Mi Perfil** (profile.html): Edición de perfil y foto de usuario
